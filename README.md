### Frpc Windows 客户端

实际上就是个壳。

编译生成`.jar` 文件，和`frpc.exe`,`frpc_min.ini` 放在同一个目录。

解决了原始`frpc.exe` 无法最小化到托盘的历史痛点（误）。
